# 🚀 Full Stack Web App

A modern full-stack web application built using FastAPI for the backend and a responsive frontend using HTML, CSS, and JavaScript. The project focuses on fast API response handling, clean architecture, and smooth frontend-backend communication for real-time interaction.

## ✨ Features

⚡ Fast and lightweight API using FastAPI  
🌐 Responsive and interactive frontend UI  
🔗 Seamless frontend–backend integration using REST APIs  
💬 Chat / request-response based system  
🧠 Structured backend routing and modular code design  
📁 Clean folder structure for scalability  
🔄 Easy to modify, extend, and upgrade  

## 🛠️ Tech Stack

Backend: FastAPI, Python, Uvicorn  
Frontend: HTML, CSS, JavaScript  
Communication: REST API (JSON based)  
Tools: Git, GitHub

## 🚀 How It Works

Frontend sends requests to FastAPI backend → Backend processes logic → Returns JSON response → Frontend updates UI dynamically based on response.

## ⚙️ Setup Instructions

Clone repository:
git clone https://github.com/dev-shivv/Assistant-Sh1v.git

Go to backend:
cd backend

Install dependencies:
pip install fastapi uvicorn

Run server:
uvicorn main:app --reload
